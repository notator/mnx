Time signatures

In MusicXML, each part’s <measure> can contain an <attributes> element with an optional <time>. MNX uses a score-wide <global> element. MNX also uses a “microsyntax” — e.g., 4/4 for common time — for brevity. In both cases, you only specify the time signature when it differs from the previous bar.
