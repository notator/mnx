Slurs (targeting specific notes)

In MusicXML, slurs are always defined within <note> elements.

In MNX, a slur can optionally specify which specific notes it’s attached to, using the start-note and end-note attributes. The start-note must be within the event that contains the <slur>, and the end-note must be within the target event.

Note that all references to an internal id consist of a '#' character followed by the id string.
