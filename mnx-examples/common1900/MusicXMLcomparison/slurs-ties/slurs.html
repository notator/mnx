Slurs

In MusicXML, you put <slur> elements within the first and last notes in the slur. The first has type=start and the last has type=stop. Use the placement attribute to specify whether the slur is above or below the notes.

In MNX, you put a single <slur> element inside the slur’s first <event>. The element’s target attribute specifies the ID of the last event in the slur. (Every <event> element can have an optional ID.) Use the side attribute to specify whether the slur is above or below.

Note that all references to an internal id consist of a '#' character followed by the id string.
