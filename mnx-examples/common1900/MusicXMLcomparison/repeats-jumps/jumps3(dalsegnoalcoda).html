Jumps3 (D.S. al Coda)

In this example, the bars are to be performed in this order: 1, 2, 3, 4, 5, 2, 3, 7.
(Measure 6 is never played.)

MusicXML uses <segno> to specify the visual segno symbol and <words> for the textual instructions. (Parsers need to interpret the text to extract semantic meaning.) For the playback instructions, MusicXML uses the <sound> element.

In MNX, all texts, segno symbols etc. are graphic objects. Performance information is coded separately. Only the graphics are coded in this example. A similar example containing performance information also exists.
Note: MNX uses custom HTML entities (&segno; and &coda;) to encode the segno and coda characters.
