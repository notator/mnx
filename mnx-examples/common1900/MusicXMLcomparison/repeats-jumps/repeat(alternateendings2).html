Repeats (with alternate endings, advanced)

In both MusicXML and MNX, if a bar has multiple endings, the <ending> element’s number attribute is a comma-separated list of the ending numbers (using one-based, not zero-based numbers).

In MNX, the attribute is the non-functional, displayed string. The order in which the bars are performed is described using explicit temporal information. (See the equivalent example that includes temporal info.)
