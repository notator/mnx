Jumps 1 (Dal Segno)

The measures are to be performed in this order: 1, 2, 3, 4, 5, 2, 3, 4, 5.

MusicXML uses <segno> to specify the visual segno symbol and <sound> elements with segno or dalsegno attributes to specify the playback instructions.

In MNX, all texts, segno symbols etc. are graphic objects. Performance information is coded separately. Only the graphics are coded in this example. A similar example containing performance information also exists.
Note: MNX uses a custom HTML entity (&segno;) to encode the segno character.
