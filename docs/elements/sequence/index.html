MNX specification

Elements > sequence

The <sequence> element

  • Parent element: <measure>
  • Child elements: <beams> <directions> <event> <grace> <tuplet>

Attributes

Name Type Required? Description
orient Orientation No Default orientation of the content in this sequence. If not provided, the orientation is determined automatically according to the implementation’s rendering rules and may be overridden by descendant elements.

Examples

This element is used in the following examples:

Accidentals, Beams, Beams (across barlines), Beams (hooks), Beams (secondary beam breaks), Beams (with inner grace notes), Dotted notes (augmentation dots), Jumps (D.S. al Fine), Jumps (Dal Segno), Key signatures, Multiple voices, Octave shifts (8va), Parts, Repeats, Repeats (more than once repeated), Repeats (with alternate endings, advanced), Repeats (with alternate endings, simple), Repeats (with implied start repeat), Slurs, Slurs (for chords), Slurs (incomplete slurs), Slurs (targeting specific notes), Three-note chord and half rest, Ties, Time signatures, Tuplets, Two-bar C major scale, “Hello world”
