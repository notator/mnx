Elements > key

The <key> element

  • Parent elements: <directions> <directions>
  • Child elements: None
A key signature applicable to this and all following measures, until changed. elements in are required for every key change. elements below are optional; if present, their "fifths" value must either be identical to the corresponding value in the global key or different by a multiple of 12.

Attributes

Name Type Required? Description
fifths Integer (signed) Yes The number of fifths distance from a signature with no accidentals.
