MNX example: “Hello world”

This basic MNX document contains a single middle C whole note, on a treble clef stave, in 4/4 time.

See also: This same example in MusicXML

<mnx> <global> <measure> <directions> <time signature="4/4"/> </directions> </measure> </global> <part> <part-name> Music </part-name> <measure barline="regular"> <directions> <clef sign="G" line="2"/> </directions> <sequence> <event value="/1"> <note pitch="C4"/> </event> </sequence> </measure> </part> </mnx>
