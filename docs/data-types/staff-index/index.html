Staff index data type

The one-based numeric index of a staff within a part. In a single-staff part, this will be "1". In a grand staff part (which has two staves), the top staff has staff index "1" and the bottom staff has staff index "2". Within a part, staff indexes should always increment by 1. For example, it's an error for a two-staff part to use staff indexes "1" and "3".
